Он поддерживает 58 глобальных точек доступа и предоставляет [домашний broadband США](https://github.com/jdnei/naiyun#1%E7%BE%8E%E5%9B%BD), [домашний broadband Гонконга](https://github.com/jdnei/naiyun#2%E9%A6%99%E6%B8%AF), [домашний broadband Тайваня](https://github.com/jdnei/naiyun#3%E5%8F%B0%E6%B9%BE), [домашний broadband Японии](https://github.com/jdnei/naiyun#4%E6%97%A5%E6%9C%AC), [домашний broadband Кореи](https://github.com/jdnei/naiyun#5%E9%9F%A9%E5%9B%BD), а также домашний broadband Малайзии.
